📊 Project Overview

About This Project

The AI Interview Coach is a real-time interview practice platform that provides live feedback on communication skills, body language, and answer quality using AI-powered analysis.

Key Features

🎥 Video Analysis

Real-time eye contact tracking, facial expression analysis, and gesture monitoring

🎤 Audio Analysis

Speech rate monitoring, filler word detection, and voice tone analysis

💬 Speech Recognition

Live speech-to-text transcription of interview answers

🤖 AI Feedback

Google Gemini LLM generates questions and evaluates answers

⚡ Real-time Interventions

Instant alerts for communication issues during the interview

📈 Analytics Dashboard

Detailed performance metrics and progress tracking

Main Actors

👤 Candidate/Interviewee

End user practicing interview skills, provides verbal and visual responses

🤖 AI Coach/System

Backend AI services that analyze performance and generate feedback

⚙️ Backend Services

Infrastructure handling WebSocket, database, and media processing

Technology Stack

Frontend: React 18, Tailwind CSS, Web Speech API, MediaRecorder API Backend: FastAPI (Python), WebSocket, SQLAlchemy Database: PostgreSQL AI/ML: Google Gemini (LLM), MediaPipe/OpenCV (Video), Librosa (Audio) Real-time: Native WebSocket for bidirectional communication
